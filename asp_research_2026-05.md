# ASP案件調査レポート｜楽楽精算・弥生会計・GMOサイン
_調査日: 2026年5月6日_

---

## 1. 弥生会計

| 項目 | 内容 |
|------|------|
| 取扱ASP | A8.net / もしもアフィリエイト / バリューコマース |
| 報酬単価 | 300円〜6,000円/件（プランによって異なる） |
| 確定条件 | 新規ユーザーからの申込み（有料入金不要） |
| 備考 | 最大1年間無料キャンペーン中は2,000円が目安。無料体験のみは300円 |

**→ アクション**: A8.netにログインし「弥生」で検索 → 承認申請する

---

## 2. 楽楽精算

| 項目 | 内容 |
|------|------|
| 取扱ASP | **公開情報では確認できず** |
| 考えられる取扱ASP | アクセストレード / afb / インフォトップ（BtoB系） |
| 報酬単価目安 | BtoB向けSaaSのため1,000〜5,000円/件が相場 |
| 備考 | ラクスが直接アフィリプログラムを運営している可能性あり。公式サイトで「パートナープログラム」を確認推奨 |

**→ アクション**: 
1. A8.net / afb / アクセストレードで「楽楽精算」を検索
2. 見つからない場合は https://www.rakurakuseisan.jp/ の問い合わせからアフィリ提携を打診
3. 代替として、比較記事では**マネーフォワードクラウド経費（A8設置済）**にCTAを向けておく

---

## 3. GMOサイン

| 項目 | 内容 |
|------|------|
| 取扱ASP | GMO系ASP（アフィタウン）/ A8.net の可能性あり |
| 報酬単価目安 | 電子契約系は1,000〜3,000円/件が相場 |
| 備考 | GMOグローバルサインのアフィリプログラムは別物。GMOサイン（電子契約）の案件は要直接確認 |

**→ アクション**:
1. A8.netで「GMOサイン」「電子サイン」で検索
2. Great Sign（レントラックス審査待ち）が承認されたら、クラウドサイン vs GMOサイン比較記事のCTAを**Great Sign**に変更
3. それまでのつなぎとして「クラウドサイン 無料で試す」CTAを設置

---

## まとめ・推奨アクション

| サービス | 即時対応 | 中期対応 |
|---------|---------|---------|
| 弥生会計 | A8.netで承認申請 → 弥生評判記事でCTA設置 | バリューコマースも並行申請 |
| 楽楽精算 | 比較記事はMFクラウド経費CTAで代替 | 直接提携打診 or A8で発見次第申請 |
